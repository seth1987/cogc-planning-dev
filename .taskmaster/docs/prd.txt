# Product Requirements Document - COGC Planning

## Projet
Application de gestion de planning pour le COGC Paris Nord

## Objectif
Corriger et améliorer l'application existante de planning COGC

## Fonctionnalités principales

### 1. Corrections urgentes
- Corriger les jours fériés (actuellement le 14 de chaque mois)
- Restaurer le module de gestion des agents
- Restaurer le module d'upload PDF
- Restaurer le module des habilitations
- Nettoyer les warnings d'imports non utilisés

### 2. Refactoring
